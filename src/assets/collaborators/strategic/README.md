Coloca aqui los logos de colaboradores estrategicos para la ruleta de la home.

Formatos admitidos:
- `svg`
- `png`
- `jpg`
- `jpeg`
- `webp`
- `avif`

Recomendaciones:
- Usa fondo transparente cuando sea posible.
- Nombra cada archivo con el nombre de la marca, por ejemplo `notaria-garcia.svg`.
- Evita espacios y caracteres raros en el nombre del archivo.

La home los carga automaticamente desde `src/pages/[lang]/index.astro`.

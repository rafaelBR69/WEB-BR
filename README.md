# WEB-BR

Web inmobiliaria multidioma (Astro) con:

- listado de propiedades
- detalle de propiedad/proyecto
- mapa interactivo con Mapbox
- pagina de proyectos
- pagina de posts
- pagina de equipo

## Start rapido

1. `npm install`
2. `npm run dev`
3. abrir `http://localhost:4321`

## Variables de entorno

En `.env`:

- `PUBLIC_MAPBOX_TOKEN`
- `PUBLIC_MAPBOX_STYLE`
